🧬 《霓虹交易所》版本演進表
v1.0 基礎交易版 (原始版本)
✨ 核心功能：建立小組資金、10年固定劇本的股市漲跌模擬、排行榜系統。

💾 存檔機制：使用瀏覽器本地端 (localStorage) 存檔。

🐛 痛點：換電腦控場時存檔會遺失，只能在同一台電腦上運作；學生只能被動看漲跌，缺乏主動介入市場的機制。

v1.1 跨機存檔更新 (解決設備限制)
✨ 新增功能：新增「📥 匯出檔案」與「📤 匯入檔案」按鈕。

🛠️ 系統改良：將遊戲進度打包成實體的 .json 檔案，老師可以用隨身碟或 LINE 將存檔帶到任何教室的電腦上無縫接軌。

v2.0 瘟疫變異版 (導入 Plague Inc. 機制)
✨ 介面大翻新：整合「新聞頭條」、「交易所」、「檔案庫」三大介面，並加入底部跑馬燈。

✨ 策略點數 (SP)：引入全新資源「SP」，每組每年固定獲得 1 點。

✨ 策略研究所：新增技能樹，可消耗 SP 發動【商業間諜】、【政府補貼】、【媒體帶風向】。

✨ 動態泡泡系統：畫面上隨機浮現「黃金泡泡」與「DNA 泡泡」，點擊可獲得大量現金或 SP，增加搶奪互動。

v2.5 歡樂大亂鬥版 (強化 PVP 與黑色幽默)
✨ 技能樹擴張 (互搞機制)：

新增 【拔網路線】(2 SP)：可隨機駭入第一名隊伍，偷走 5 萬元現金。

新增 【董事長跑路】(3 SP)：指定股票年底強制暴跌 20%。

強化 【國安基金護盤】(3 SP)：跌幅由系統全額補貼退回現金。

✨ 黑色幽默跑馬燈：在正規新聞跑馬燈中，混入「高中生數學及格被譽為奇蹟」、「投資靠丟飛鏢」等瘟疫公司風格的假新聞。

⚠️ 陷阱泡泡機制：新增紅色的「查稅局泡泡」，點錯會導致該組被重罰 5 萬元，考驗動態視力。

v3.0 情報戰升級版 (強化資訊不對稱)
✨ 持股情報網 (新介面)：新增第四個面板「👁️ 持股情報網」，老師可切換至此畫面，讓全班看到各組目前的「手邊現金」以及「買了哪些股票（僅顯示代號與模糊提示）」，增加心機攻防。

🛠️ 系統改良 (最高機密彈窗)：將【內線交易】技能從「公開顯示在新聞上」改為「黑色駭客風格的獨立彈窗」。老師可以偷偷叫該組學生上前觀看，看完點擊「閱畢銷毀」，實現真正的資訊不對稱。

🛠️ 系統改良：為老師新增在「隊伍明細」視窗中「一鍵全數賣出」單一股票的按鈕，加快控場結算速度。

v3.1 平衡與極限修復版 (防呆與經濟平衡) —— 📍 當前最新版
⚖️ 經濟平衡調整 (泡泡 Nerf)：將點擊金錢與陷阱泡泡的金額，從「影響戰局的數萬元」大幅下修為「1~3 元的微小零錢/微小虧損」。保留了點擊的舒壓感與趣味性，但確保遊戲勝負依然建立在學生的「數學計算與資產配置」上，維護遊戲公平性。

🐛 Bug 修復 (情報重複問題)：修正【內線交易】連續使用時會抽到同一條新聞的問題。現在系統會記憶該組看過的情報，保證每次抽都看到不同的新聞。

🐛 防呆機制 (情報枯竭)：新增情報探底限制。如果該年度明年的新聞已經被某組全部偷看光了，再次點擊時會跳出「情報已被看光」的警告，並且不會扣除該組的 SP，防止學生浪費點數。
